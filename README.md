

# Run and deploy

Windows prerequisite:
   - docker desktop
Linux prerequisite:
   - docker engine
   - docker compose
   
Go to project directory and use "docker-compose -f docker-compose.yml up --build -d" or "docker compose up --build -d"

## Run Locally 

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
